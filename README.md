# Housing Price Prediction using Linear Regression
 Project Overview

This project aims to predict house prices using Multiple Linear Regression based on various property features such as area, number of bedrooms, bathrooms, stories, parking availability, and furnishing status.

The goal is to analyze how different housing features impact the final property price and build a predictive model.

# Problem Statement

To build a regression model that predicts house prices based on property features.

1. Data Preprocessing
